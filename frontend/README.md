### Установка pnpm
```sh
wget -qO- https://get.pnpm.io/install.sh | sh -
```
### Создание react приложения
```sh
pnpm create vite webapp --template react-ts
```
### Установка пакетов из файла package.json
```sh
pnpm i
```
### Запуск для разработки
```sh
pnpm dev
```
### Сборка
```sh
pnpm build
```
### Запуск собранного
```sh
pnpm preview
```
